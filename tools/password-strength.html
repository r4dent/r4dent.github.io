Common Password Mistakes

  • Using personal information (names, birthdays)
  • Using common word patterns (password123)
  • Using keyboard patterns (qwerty, 12345)
  • Reusing passwords across multiple accounts
  • Using simple character substitutions (p@ssw0rd)
