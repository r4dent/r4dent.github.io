Win + R Commands

compmgmt.msc

Opens the Computer Management console, which provides access to a range of system tools and resources, such as the Device Manager, Disk Management, and the Event Viewer.

devmgmt.msc

Opens the Device Manager, which allows you to view and manage the hardware devices on your computer.

diskmgmt.msc

Opens the Disk Management console, which allows you to manage hard disk drives and partitions on your computer.

eventvwr.msc

Opens the Event Viewer, which allows you to view system and application logs on your computer.

services.msc

Opens the Services manager, which allows you to view and manage the services on your computer.

secpol.msc

Opens the Local Security Policy editor, which allows you to view and configure security settings on your computer.

netplwiz

Opens the User Accounts window, which allows you to manage user accounts and settings on your computer.

lusrmgr.msc

Opens the Local Users and Groups manager, which allows you to manage user accounts and groups on your computer.

taskmgr

Opens the Task Manager, which allows you to view and manage running programs and processes on your computer.

sysdm.cpl

Opens the System Properties window, which allows you to view and manage system-wide settings and configurations on your computer.
