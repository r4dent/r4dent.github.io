Welcome to IT Tools

Free, open-source utilities for IT professionals, security researchers, and developers.

Privacy First

All tools run entirely in your browser. No data is ever sent to any server or stored anywhere.

Networking Tools

Professional networking utilities including subnet calculator, IP geolocation, DNS lookup, and more. Perfect for network administrators and IT professionals.

Security Tools

Essential security tools including password generation, hash calculations, and encoding utilities. Designed for security professionals and privacy-conscious users.

Development Tools

Helpful development utilities including formatters for JSON and XML, plus a powerful regex tester. Built for developers and technical professionals.
